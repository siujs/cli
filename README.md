# siu cli

#### Description

Extensible front-end application scaffold, sounds like football moving: `siu...`

## Document

Docs area avaliable at [Siu](https://siujs.github.io/cli/)

### Running Tests

To run test on all packages:

```console
$ yarn test
```

To run tests on a specific package:

```console
$ yarn test <name>
```
